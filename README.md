Roblox vs Minecraft Text Analysis

Mini project ini mengeksplorasi **perbedaan karakteristik teks** antara dua game populer: **Roblox** dan **Minecraft** melalui pendekatan **Web Scraping + Natural Language Processing (NLP)**.

## Highlight

*  Mengumpulkan data teks secara otomatis (web scraping)
*  Melakukan preprocessing teks (cleaning, tokenization, stopword removal, stemming)
*  Menganalisis frekuensi kata untuk menemukan pola perbedaan
*  Visualisasi hasil dalam bentuk **wordcloud** yang intuitif

##  Insight

Project ini menunjukkan bagaimana data teks tidak terstruktur dapat diolah menjadi informasi yang bermakna, serta mengungkap **perbedaan fokus/topik pembahasan** antara Roblox dan Minecraft berdasarkan kata yang dominan muncul.

##  Tech Stack

* Python
* Pandas, NLTK / Sastrawi
* BeautifulSoup / Requests
* Matplotlib, WordCloud

##  Output

* Dataset hasil scraping
* Teks yang sudah diproses
* Wordcloud Roblox vs Minecraft

##  Value

Project ini merepresentasikan kemampuan dalam:

* Data Collection (Scraping)
* Data Preprocessing (NLP)
* Data Analysis & Visualization
